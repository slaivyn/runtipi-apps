# OpenProject

OpenProject is a powerful open-source project management platform that supports classic, agile, and hybrid project management methodologies. Built with data sovereignty and security in mind, it provides teams with comprehensive tools for planning, tracking, and collaborating on projects.

## Key Features

- **Project Planning**: Create detailed project schedules with interactive Gantt charts, timeline views, and milestone tracking
- **Agile Support**: Full Scrum and Kanban board implementation with sprint planning, backlogs, and velocity tracking
- **Task Management**: Centralized work package tracking with custom fields, statuses, and workflows
- **Team Collaboration**: Real-time collaboration, meeting protocols, and transparent communication across teams
- **Time & Cost Tracking**: Built-in time logging, budgeting, and cost reporting capabilities
- **Team Planner**: Visual workload distribution and capacity planning with calendar views

## Use Cases

OpenProject is ideal for:
- Software development teams using Agile/Scrum methodologies
- Project portfolio management across multiple teams
- Construction and BIM project coordination
- Product roadmap and release planning
- Educational and research institutions
- Public sector organizations requiring data sovereignty

## Security & Privacy

- Self-hosted solution ensuring complete data control
- HTTPS encryption and secure session management
- Two-factor authentication support
- GNU GPL v3 licensed for transparency

## Default Credentials

After installation, log in with:
- **Username**: `admin`
- **Password**: `admin`

**Important**: Change the default password immediately after first login.

## Getting Started

Once deployed, OpenProject will be accessible through your Runtipi domain. The initial setup wizard will guide you through basic configuration including language preferences, project creation, and user management.
